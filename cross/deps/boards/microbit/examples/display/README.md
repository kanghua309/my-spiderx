# microbit-bsp-display-example

Demonstrating the use of the LED matrix on the BBC micro:bit.


## Prerequisites

Software:

* [`rustup`](https://rustup.rs/)
* [`probe-run`](https://github.com/knurling-rs/probe-run)

Hardware:

* [BBC micro:bit v2](https://microbit.org/)

## Running

```
cargo run --release
```
